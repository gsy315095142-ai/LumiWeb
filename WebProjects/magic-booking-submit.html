提交与跳转

表单校验

点击「下一步」前，系统校验以下条件：

  • 必须选择日期
  • 必须选择时间点
  • 必须选择场次
  • 必须选择人数（新建预约时）
  • 必须已登录（未登录弹出手机号授权）

跳转确认页（新建预约）

校验通过后，携带以下参数跳转到 /pages/confirm/confirm：

参数说明
place门店 ID
date选择的日期
num预约人数
session选中场次的完整 JSON 对象
bookStartTime进入预约页的时间戳（用于统计）

直接修改场次（改签模式）

如果是修改已有订单，选择新场次后直接调用 /mz/changeGameRoundForOrder，传入 order_id 和 round_id，成功后返回上一页。
