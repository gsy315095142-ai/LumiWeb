角色创建页（role）

页面入口

角色创建页（/pages/role/role）通过激活码激活成功后跳转进入，携带参数：order（订单 ID）和 status（订单状态）。订单 ID 为 0 时直接返回。

加载角色列表

进入页面后调用 /mz/getGameRoleList，获取该订单下的所有角色。每个角色包含：

字段说明
id角色 ID
nickname角色名称（初始为空）
gender性别（0=女，1=男）
created_by创建者用户 ID
actived_by激活者用户 ID
status角色状态

角色数量 = 预约人数。每个角色根据 gender 匹配对应的形象图片（CDN 地址）。

选择角色形象（性别）

每个角色支持切换性别形象：

  • 默认提供两个形象：女性角色、男性角色
  • 形象图片存放在 CDN：cdn.lixiaoqian.com/magic-space/roles/female.png / male.png
  • 通过左右箭头切换，已到边界时不可继续切换
  • 切换形象同时修改角色的 gender 值

命名角色

为每个角色输入名称：

  • 字符限制：中文最多 6 个字符，不可输入符号（自动过滤）
  • 重复检测：同一订单内不可有重名角色
  • 随机取名：点击骰子按钮随机生成名称（如「聪慧的小狐狸」「勇敢的小老虎」「机灵的小花猫」等）
  • 点击不同角色卡片切换当前编辑角色

提交角色信息

点击确认后进行以下校验：

  • 所有角色名称不能为空
  • 角色名称不可重复
  • 角色名称不可包含特殊符号
  • 必须有实际修改才可提交（否则提示「您暂无修改」）

校验通过后调用 /mz/submitGameRoles，传入 order_id 和角色数组。成功后进入倒计时视图。
