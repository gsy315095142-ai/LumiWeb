➕ 新增优惠活动

/pages/promotion/promotion，activeTab='add' 模式。

  • 三 Tab：新增/列表/统计，蓝色下划线选中态 2B84FD 6rpx，CDN 深色背景
  • 活动信息卡片：白底 border-radius:26rpx，box-shadow，含 4 个表单项
  • 活动条件：picker 下拉，从 /mz-admin/promotionPageOpts 动态加载（不限制条件/首单赠送/…）
  • 赠券次数：每次赠送 N 张，picker 选择
  • 优惠券信息卡片：优惠券名称（≤6字）/ 形式 / 折扣值 / 每日限制 / 转赠开关
  • 优惠券形式：按比例折扣(%) / 固定金额折扣(元) / 满减(元)，选择「满减」时额外显示「满足金额」字段
  • 保存：/mz-admin/createPromotion，编辑时走 /mz-admin/savePromotion
  • 输入验证：中文字符算 1，英文算 0.5；折扣比例 ≤ 100%；减免 ≤ 满足金额

📋 活动列表管理

activeTab='list' 模式。

  • 搜索栏：按活动名称搜索，含清除按钮，debounce 防抖
  • 活动卡片：头部 linear-gradient(0deg, #fff, #F3F9FF) 渐变 + 蓝色标题，左 ✏️ 编辑 / 右 🗑️ 删除
  • 生效门店：multi-select 多选组件，已启用活动灰色不可操作
  • 活动期限：开始/结束日期，点击弹出自定义 picker-view（年月日时分 5 列滚动）
  • 优惠券期限：固定日期（日区间选择）/ 固定天数（输入 N 天），单选切换
  • 启用/禁用开关：自定义滑块 custom-switch，启用=绿 #32DE90，禁用=灰 #e5e5e5
  • 保存：二次校验（门店/日期/有效期）→ /mz-admin/updatePromotionStatus
  • 编辑：点击 ✏️ → /mz-admin/promotionDetail → 回填到 add 表单
  • 删除：确认弹窗 → /mz-admin/deletePromotion（已启用不可删除）

📊 活动统计

activeTab='stats' 模式。

  • 日期搜索：search-date 子组件，默认当月，结束日期限制为当月最后一天
  • 查询条件：酒店（全部/指定门店）→ 优惠活动名称 → 优惠券名称，级联筛选
  • 数据网格：grid-template-columns: repeat(2, 1fr) 两列布局，6 项统计指标
  • 统计指标：发放数量 / 过期数量 / 使用数量 / 对应票券数量 / 已核销未退票数量 / 已核销未退票人数
  • 导出：右下角浮动按钮（需 view-stat-hotel 权限），wxFileDownload 导出 Excel
  • 数据来源：/mz-admin/promotionStats，参数：日期范围 + store_id + promotion_id + promo_name

🎟️ 用户端优惠券

/pages/coupons/coupons，CDN 深色背景，scroll-view 滚动加载。

  • 卡片样式：白底 border-radius:16rpx，box-shadow，margin:20rpx
  • 状态标签：待使用=蓝 #e6f7ff / #1890ff，已使用=绿 #f6ffed / #52c41a，已过期=灰 #f5f5f5 / #8c8c8c
  • 标题行：优惠券名称 + 适用人数（如"2人"）
  • 时间信息：兑换日期 + 有效期至（已使用显示使用日期）
  • 剩余天数：仅待使用状态显示，color:#faad14 橙色，带日历图标
  • 滚动加载：bindscrolltolower 触底加载更多，lower-threshold:100
  • 状态筛选：通过 URL 参数 type 过滤（0=全部 / 1=待使用 / 2=已使用 / 3=已过期）
