🔐 登录页主界面

页面路径：/pages/login/login。整体为全屏背景大图设计（login-bg.jpg），背景图撑满整个屏幕。

  • 背景：全屏大图，通过 CSS background-size: auto 100% 纵向撑满
  • 微信登录按钮：固定在底部（position: fixed; bottom: 260rpx），实际是一张图片（login-weixin.png）覆盖在透明按钮上，点击触发 handleToIndex
  • 协议勾选：固定在页面最底部（position: fixed; bottom: 150rpx），圆形勾选框 + 文案，点击切换 agreeDeal 状态

进入流程：进入页面 → 自动调用 wxLogin() 获取临时 Token → 勾选协议 → 点击微信登录按钮 → 将 tem_token 转存为 access_token → reLaunch 跳转首页
