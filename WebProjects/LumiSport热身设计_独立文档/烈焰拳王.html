烈焰拳王 · 热身设计

示意图 交互原型

🎯热身目标：让玩家在正式对战前掌握出拳、格挡、闪避三大核心操作，并通过一次完整连招体验节奏感。

1. 热身步骤总览

步骤名称核心动作通过条件原型
步骤 1出拳前手直拳 / 后手直拳命中沙袋 3 次示意图
步骤 2格挡双手举至面部前方成功格挡 2 次示意图
步骤 3闪避身体左 / 右侧倾成功闪避 2 次示意图
步骤 4连招练习出拳 → 格挡 → 闪避 → 重拳完成 1 套连招交互

→ 步骤之间为顺序锁定：完成当前步骤后自动进入下一步。

2. 步骤详细设计

步骤 1：出拳

🎯 目标
学会最基本的攻击动作——前手直拳和后手直拳。
📝 VR 场景描述
玩家面前出现一个红色沙袋，悬浮在击打距离内。沙袋正面画有两个圆形靶标（左 / 右），颜色醒目。顶部提示板文字：「用你的左右手分别击打沙袋上的靶标，命中 3 次即可通过！」
▲ 步骤 1：出拳训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明；靶标发光闪烁引导注意
2. 确认双手伸出就位（确认手势）提示板消失，靶标变为可击打状态
3. 击打用左手 / 右手分别击打靶标命中时：靶标爆裂特效 + 计数更新（1/3, 2/3, 3/3）
4. 完成命中 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 2

步骤 2：格挡

🎯 目标
学会防守动作——将双手举至面部前方形成防御姿态。
📝 VR 场景描述
沙袋消失，面前出现一个训练机器人（半透明蓝色轮廓）。机器人会缓慢向玩家投出拳击（拳头前方有红色光晕预警）。顶部提示板文字：「将你的双手举到面前来格挡机器人的攻击！」
▲ 步骤 2：格挡训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，机器人开始第一拳（3 秒延迟）
3. 格挡在拳头到达前将双手举至面前成功：蓝色盾牌特效 + 清脆金属音效；失败：红色冲击波 + 提示「再试一次」
4. 完成成功格挡 2 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 3

步骤 3：闪避

🎯 目标
学会通过身体侧倾来躲避攻击。
📝 VR 场景描述
训练机器人继续存在，但攻击方式改变：机器人会水平发出一道光带（类似光墙），覆盖玩家正面。光带出现前 1 秒有预警箭头指示方向（左 / 右）。顶部提示板文字：「向左或向右倾斜身体来躲避光带！」
▲ 步骤 3：闪避训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一次光带（3 秒延迟）
3. 闪避根据预警箭头向左 / 右侧倾身体成功：光带穿过时玩家身体呈半透明（安全）；失败：红色冲击 + 提示「再试一次」
4. 完成成功闪避 2 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 4

步骤 4：连招练习

🎯 目标
将出拳、格挡、闪避组合成一套完整连招，体验实战节奏。
📝 VR 场景描述
场景恢复为正式比赛擂台氛围（背景变亮）。训练机器人变为完整实体形态，会依次发起：攻击（需格挡）→ 攻击（需闪避）→ 露出弱点（需出拳）→ 最终重拳。右侧 HUD 显示连招进度条：「出拳 → 格挡 → 闪避 → 重拳」。顶部提示板文字：「现在把所有动作组合起来，完成一套连招！」
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，机器人进入准备姿态
3. 出拳击打机器人弱点（高亮区域）命中反馈 + 进度条第 1 段亮起
4. 格挡机器人反击 → 玩家双手举至面前格挡成功反馈 + 进度条第 2 段亮起
5. 闪避机器人攻击 → 玩家侧倾闪避成功反馈 + 进度条第 3 段亮起
6. 重拳击打机器人暴露的弱点慢动作特写 + 大幅冲击特效 + 进度条全部亮起
7. 完成完成 1 套连招白闪 0.6 秒 → ✅ → 暂停 2 秒 → 循环回步骤 1（10 秒无操作）

3. 视觉风格参考

VR 第一人称视角 + 游戏引擎实时渲染；视觉基调类似《Creed: Rise to Glory》的擂台氛围 + 赛博朋克轻量级 UI 叠层；火焰特效克制不浮夸，服务于功能引导而非纯装饰；核心设计原则：所有视觉效果必须服务于信息传达和操作引导，不分散玩家注意力。
