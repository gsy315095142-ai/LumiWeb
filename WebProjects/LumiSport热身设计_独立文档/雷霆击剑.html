雷霆击剑 · 热身设计

示意图 交互原型

🎯热身目标：让玩家掌握刺击、格挡、位移三大核心操作，理解攻击窗口（金色暴露区域）与护盾机制。

1. 热身步骤总览

步骤名称核心动作通过条件原型
步骤 1刺击光剑直线前刺命中目标命中 3 次示意图
步骤 2格挡光剑横挡格挡来袭攻击成功格挡 2 次示意图
步骤 3位移前/后/左/右移动至指定区域到位 3 次示意图
步骤 4综合练习格挡 → 位移 → 寻找窗口 → 刺击完成 1 套完整攻防交互

→ 步骤之间为顺序锁定：完成当前步骤后自动进入下一步。

2. 步骤详细设计

步骤 1：刺击

🎯 目标
学会最基本的攻击动作——用光剑直线前刺命中目标。
📝 VR 场景描述
玩家手持一把蓝色光剑，面前出现一个训练人偶。人偶胸口有一个金色圆形靶标（暴露区域），周围是蓝色护盾覆盖。顶部提示板文字：「用你的光剑刺击人偶胸口的金色区域，命中 3 次即可通过！」
▲ 步骤 1：刺击训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明；金色靶标脉冲闪烁
2. 确认双手伸出就位（确认手势）提示板消失，靶标变为可攻击状态
3. 刺击挥动光剑刺向金色靶标命中：电光碰撞特效 + 计数（1/3）；刺到护盾：蓝色火花弹开 + 提示「瞄准金色区域」
4. 完成命中 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 2

步骤 2：格挡

🎯 目标
学会用光剑格挡对手的攻击。
📝 VR 场景描述
训练人偶现在手持一把红色光剑，会缓慢向玩家发起直线刺击（有红色预警线提示）。顶部提示板文字：「将你的光剑横在身前来格挡对手的攻击！」
▲ 步骤 2：格挡训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，人偶准备攻击（3 秒延迟）
3. 格挡在红色预警线消失前将光剑横挡在身前成功：光剑碰撞电光特效 + 清脆金属音效；失败：红色冲击波 + 提示「再试一次」
4. 完成成功格挡 2 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 3

步骤 3：位移

🎯 目标
学会在地面上快速移动到指定位置，理解位移在攻防中的作用。
📝 VR 场景描述
地面出现一个发光圆环（金色），提示玩家移动过去。圆环会在不同位置依次出现（前/后/左/右）。顶部提示板文字：「移动到地面上的金色圆环内，到位 3 次即可通过！」
▲ 步骤 3：位移训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一个圆环出现
3. 位移走到/侧步到圆环内到位：圆环爆裂金色特效 + 计数（1/3）；超时（5 秒）：提示「快移动到圆环内」，圆环重置
4. 完成到位 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 4

步骤 4：综合练习

🎯 目标
将格挡、位移、刺击串联为完整攻防回合。
📝 VR 场景描述
场景升级为正式击剑赛道。训练人偶变为完整实体对手，会发起攻击并暴露弱点。右侧 HUD 显示流程进度条：「格挡 → 位移 → 寻找窗口 → 刺击」。顶部提示板文字：「把所有动作组合起来，完成一次完整攻防！」
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，对手进入准备姿态
3. 格挡对手攻击 → 玩家光剑横挡格挡成功电光特效 + 进度条第 1 段亮起
4. 位移对手后退露出窗口 → 玩家移至攻击位到位确认 + 进度条第 2 段亮起
5. 刺击对手金色暴露区域出现 → 玩家刺击命中电光爆裂特效 + 进度条全部亮起
6. 完成完成 1 套攻防白闪 0.6 秒 → ✅ → 暂停 2 秒 → 自动循环回步骤 1

3. 视觉风格参考

VR 第一人称视角 + 游戏引擎实时渲染；视觉基调类似《Beat Saber》的光剑美学 + 击剑赛道竞技感；光剑碰撞的电光特效为核心视觉记忆点；金色暴露区域与蓝色护盾的色彩对比要清晰可辨，确保玩家在快速攻防中一眼识别攻击窗口；核心设计原则：攻防状态即时可辨、光剑轨迹清晰、位移方向指示明确不迟疑。
