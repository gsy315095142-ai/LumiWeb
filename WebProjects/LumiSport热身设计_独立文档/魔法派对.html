魔法派对 · 热身设计

示意图 交互原型

🎯热身目标：让玩家掌握左/右手识别、元素抓取、节奏打击三大核心操作，理解魔法元素从远到近的飞行轨迹。

1. 热身步骤总览

步骤名称核心动作通过条件原型
步骤 1左右手识别用对应颜色的手触碰飞行元素正确命中 3 次示意图
步骤 2元素抓取在正确时机伸手抓取元素成功抓取 3 次示意图

→ 步骤之间为顺序锁定：完成当前步骤后自动进入下一步。

2. 步骤详细设计

步骤 1：左右手识别

🎯 目标
学会区分左手和右手——魔法派对中元素会标注需要用哪只手接。
📝 VR 场景描述
玩家身处霓虹魔法舞台中央。远处有元素发射器，会依次向玩家发射彩色光球。每个光球带有明确的左/右标识：左侧蓝色光球需用左手触碰，右侧红色光球需用右手触碰。顶部提示板文字：「看准光球的颜色，用对应的手去触碰它！」
▲ 步骤 1：左右手识别训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一颗光球发射（3 秒延迟）
3. 触碰用对应颜色的手触碰飞来的光球正确：光球爆裂为对应颜色烟花特效 + 计数（1/3）；错误：光球弹开 + 提示「注意颜色对应的手」
4. 完成正确命中 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 2

步骤 2：元素抓取

🎯 目标
学会在正确时机伸手抓取从远处飞来的魔法元素。
📝 VR 场景描述
发射器现在发出完整的魔法元素（火球、冰晶、风刃），元素沿 3.5 秒飞行轨迹从远处飞向玩家。元素到达最佳抓取位置时会有光圈脉冲提示。顶部提示板文字：「等元素飞到最近处时伸手抓住它！」
▲ 步骤 2：元素抓取训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一个元素发射
3. 抓取在元素到达最近处时伸手触碰成功：元素吸附到手中 + 对应属性光芒特效 + 计数（1/3）；太早/太晚：元素穿过 + 提示「注意光圈提示的时机」
4. 完成成功抓取 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 3

步骤 3：节奏打击

🎯 目标
学会按节奏连续击打飞来的元素，体验魔法派对的核心玩法。
📝 VR 场景描述
元素开始按固定节拍（约 1.2 秒间隔）从远处依次飞来，形成 3 条飞行轨迹（左/中/右）。每个元素带有左/右手标识和颜色。背景音乐开始播放，节拍与元素飞行同步。右侧 HUD 显示连击计数器。顶部提示板文字：「按节拍用对应的手击打飞来的元素，连续命中 5 个！」
▲ 步骤 3：节奏打击训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，节拍倒计时 3 拍后开始
3. 打击按节拍用对应手击打飞来的元素命中：烟花爆裂 + 连击计数 +1（Perfect/Good）；漏击：连击中断归零 + 提示「继续，重新计数」
4. 完成连续命中 5 个白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 4

步骤 4：综合练习

🎯 目标
将识别、抓取、节奏打击串联为完整体验。
📝 VR 场景描述
场景升级为正式魔法派对舞台，霓虹灯效全开。元素以更高频率和更复杂轨迹飞来，同时混合不同属性和左右手标识。右侧 HUD 显示完整流程进度。顶部提示板文字：「现在是真正的挑战——尽情享受魔法派对吧！」
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，音乐响起，第一波元素发射
3. 连续打击识别左右手 → 抓取元素 → 按节奏连续击打 8 个每次命中：连击计数 + 评级动画（Perfect/Good/Miss）
4. 完成完成 1 套完整流程（累计 8 个有效命中）白闪 0.6 秒 → ✅ → 暂停 2 秒 → 自动循环回步骤 1

3. 视觉风格参考

VR 第一人称视角 + 游戏引擎实时渲染；视觉基调类似《Beat Saber》的光效节奏感 + 《Synth Riders》的霓虹光效语言；背景 BGM 节拍与视觉脉冲同步，增强节奏感；核心设计原则：元素从远到近的 3.5 秒飞行轨迹清晰、左右手区分一目了然、评级反馈即时且富有满足感。
