📱 手机号授权（新用户）

未登录用户执行需要身份的操作时，弹出 phone-btn 组件。新用户（is_new = true 或 phone == ''）显示「手机号快捷登录」蓝色按钮。

🛡️ 管理端权限判断

管理员（userType > 0）首页右上角显示圆形「管理」按钮，点击跳转逻辑：

  • 中控管理员（userType == 3 且非自定义角色）→ /pages/central/central
  • 其他管理角色 → /pages/ticket/ticket
  • 员工角色：store_staff 数组中是否有当前门店的管理权限
  • 超级管理员：super_man = true 时拥有所有权限
  • 功能权限：通过 getUserPermissions() 检查具体功能权限，无权限的操作按钮隐藏或提示「无操作权限」
