数据加载流程 & API

数据加载流程

步骤触发说明
1. 初始化onShow读取 placeLists（轮询 100ms），初始化酒店列表 + 用户权限
2. 选日期search-date 组件返回 ticketDates[]（起止日期），触发 handleGetStatData
3. 选酒店onHotelChange更新 selectedHotel，触发 handleGetStatData
4. 请求数据handleGetStatData参数：date_start / date_end / store_id
5. 无权限非 view-stat-hotelstoreId 取 getItem('currentStoreId')（当前用户所属门店）
6. 渲染接口返回设置 totalStatData + newStatsData + dataLoaded=true
7. 展开/收起handleShowPackage切换 expanded，显示 packages[] 套餐包明细

API 接口

接口方法参数用途
/mz-admin/getStoreDataStatsPOSTdate_start / date_end / store_id获取统计卡片数据（汇总 + 分项）
/mz-admin/downloadStatementsGETdate_start / date_end / store_id导出对账单 Excel

newStatsData 分项卡片字段

字段含义显示条件
name卡片标题（如"酒店票券统计"）始终
nickname主数据行标题（如"酒店订场票券"）始终
ticketNum / ticketValue数量（张）/ 人数始终
receiveNum已领未下单字段存在时
receiveNoOrderNum已领未下单(重复)字段存在时
refundNum已退票数量字段存在时
overdueNum已过期数量字段存在时
qrcodeNum二维码已扫码字段存在时
qrcodeInvalidNum二维码已失效字段存在时
packageRefundNum套餐退票数量字段存在时（值>100000 缩小字号）
packageNum套餐数量字段存在时
packages[]套餐包明细（name/num/value）字段存在时，展开/收起

权限说明

权限 Key可见内容
view-stat-hotel酒店选择器 + 导出按钮（管理员/老板角色）
无此权限自动使用 currentStoreId 查询本门店数据，不显示酒店选择器
