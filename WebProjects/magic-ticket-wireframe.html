👥 管理端 — 团购票生成

选择「团购票」套餐后，表单切换为团购票模式：

  • 团购票数量：数字输入框（groupPlayerNum），需 ≥ minPlayerNum（最小人数）
  • 团购票档位：自动根据人数匹配价格（groupPrice），只读
  • 团队信息：浅蓝底圆角框，显示团队名称、预约日期、预约场次
  • 选择场次：蓝色按钮 → 弹出场次选择弹窗（二维表格）
  • 按钮文案改为「申请生成团购票」→ 需管理端审批

📋 管理端 — 票券列表

pageType == 'lists'，支持筛选 + 统计 + 票券卡片列表。

  • 筛选区：日期范围选择器 + 员工筛选 + 酒店筛选 + 「仅待审核」复选框
  • 统计卡片：可折叠，显示各类票券的生成/领取数量和人数（蓝色大字 Impact 字体）
  • 票券卡片：渐变色标题栏（颜色按状态变化）+ 票券详情 + 操作按钮
  • 状态颜色：蓝(未领取) → 绿(已获取) → 橙(退票审核中) → 红(已退票) → 灰(已过期)
  • 左滑退票：状态 0/1/5 时可左滑出现红色「申请退票」按钮
  • 红点提示：Tab 上显示退票审核红点（refundAuditRedDot）
  • 支持分页加载（滚动到底部自动加载更多）
