📱前端组件详解

弹幕组件 SessionDanmakuLayer.vue 挂载在赛事主页（event.vue）中，包含两个区域：

🎬 飘屏区域（dm-fly-layer）

  • 两种定位模式 — flyAnchorInline=true 时以 absolute 铺满父级赛事模块锚点，仅覆盖 VS/赔率卡区域，不挡底部 Tab；flyAnchorInline=false（默认）时以 fixed 定位在导航栏下方
  • 3 泳道轮询分配 — 每条新弹幕按 laneRound % 3 分配到 0/1/2 泳道，垂直间距 40rpx，避免重叠
  • 随机滑速 — 每条弹幕的 animationDuration 为 11 + random(5) 秒，即 11~16 秒不等，营造自然错落感
  • 自动清理 — 每条弹幕 16 秒后自动从数组中移除；数组上限 16 条，超出时裁剪最早的
  • 文字样式 — 白色文字 + 黑色阴影（text-shadow），确保在各种背景上可读；字号 24rpx，不干扰主要赛事信息
  • pointer-events: none — 飘屏层不拦截触摸事件，不影响下方赛事模块的正常交互

⌨️ 底部输入栏（dm-input-bar）

  • fixed 固定底部 — 输入栏始终贴在屏幕底部（z-index: 95），适配安全区（safe-area-inset-bottom）
  • 输入框 — maxlength="80"，placeholder 为「发条弹幕…」，confirm-type="send" 支持键盘确认键发送
  • 发送按钮 — 发送中（sending=true）或输入为空时禁用；发送成功后自动清空输入框
  • 青蓝配色 — 白色背景 + 蓝色聚焦边框 + 渐变发送按钮，与浅色主题统一
