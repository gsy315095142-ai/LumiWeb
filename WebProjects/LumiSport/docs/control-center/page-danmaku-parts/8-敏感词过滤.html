🛡️敏感词过滤系统

后端 SensitiveWordFilterService 在服务启动时自动加载敏感词库，对弹幕内容做子串匹配：

📁
文件词库
通过环境变量 DANMAKU_SENSITIVE_WORDS_FILE 指定词库文件路径，默认使用 config/danmaku-sensitive-words.txt，每行一个词，# 开头为注释行。
🔧
环境变量词库
通过 DANMAKU_SENSITIVE_WORDS 逗号分隔传入敏感词，与文件词库合并去重。
🔍
子串匹配 + 大小写不敏感
对弹幕正文做子串包含检查（非全文精确匹配）。ASCII 字母忽略大小写，中文精确匹配。词库按长度降序排列，优先匹配长词减少误报。
🚫
命中即拒绝
命中敏感词时返回 HTTP 400，提示「弹幕内容包含敏感词，请修改后重试」，弹幕不会被存入数据库。

⚠️ 词库为空时不拦截

如果既未配置文件词库也未配置环境变量词库，服务启动时打印警告：「弹幕敏感词库为空，将不拦截任何词」。此行为是为了方便本地开发调试，生产环境务必配置词库。可通过将 DANMAKU_SENSITIVE_WORDS_FILE 显式设为空字符串来禁用文件词库。
