🔑 微信登录 · 唯一登录方式
系统仅支持 微信小程序登录 这一种登录方式。所有用户（普通用户和管理员）都通过微信一键登录进入系统。管理员身份不在登录环节区分，而是在登录后通过白名单机制自动判定。
💬
微信一键登录 🔑 唯一入口
适用角色：所有用户（user + admin）
调用 uni.login({ provider: 'weixin' }) 获取微信临时 code，发送到后端 POST /api/auth/login-wechat，后端通过 jscode2session 换取 OpenID，自动查找或创建用户，并查询 store_admins 白名单表 判定角色。用户完全无感知，无需输入任何信息。
1用户点击「微信登录 🎮」按钮
2调用 uni.login({ provider: 'weixin' }) 获取临时 code
3POST /api/auth/login-wechat { code } → 后端换取 openid
4后端查找/创建用户 → 查询 store_admins 白名单 → 判定角色
5签发 JWT（含 role + storeIds）→ 返回 { accessToken, user, managedStores }
6前端持久化 token + userInfo + managedStores → reLaunch 跳转至选择门店页
