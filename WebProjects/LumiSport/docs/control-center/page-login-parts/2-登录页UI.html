🎨 登录页 UI 元素
小程序 · 登录页
登录页采用赛博朋克视觉风格：深色底 + 青色/品红霓虹光晕，卡片绝对居中。页面仅保留微信登录一个入口，所有用户统一从此进入：
✨
品牌标题 装饰
「LumiSportSpace」文字，带青白渐变溜光动画 + 发光投影，固定展示不可交互
🎮
玩法图标行 装饰
4 个圆形图标：🥌 疾速冰球、🥊 烈焰拳王、🤺 雷霆击剑、✨ 魔法派对，每个图标有独立的主题色光环，仅展示不可交互
💬
微信登录按钮 所有用户 唯一入口
绿色渐变主按钮「微信登录 🎮」，点击触发 uni.login → 获取临时 code → 后端换 token 流程。管理员与普通用户使用同一按钮
⚠️
全局错误提示 反馈
登录失败时底部显示红色提示条（如「微信登录失败」「服务端未配置微信小程序 AppID/AppSecret」），自动清除
🛡️
角色自动判定 白名单 后台
登录成功后，后端自动查询 store_admins 白名单表，命中则返回 role=admin + managedStores 门店列表，前端无需手动选择角色
