📡 相关 API 接口
接口方法说明鉴权
/api/auth/login-wechatPOST微信登录：传入 { code }，返回 { accessToken, user, managedStores }否
/api/auth/profileGET获取当前用户资料（含门店列表），失败则触发退出登录JWT
/api/auth/profilePUT更新用户资料（昵称），角色/门店不可通过此接口修改JWT
/api/player/check-inPOST每日签到：领取 +100 竞猜币，Asia/Shanghai 自然日限一次JWT
/api/admin/store-adminsGET查询白名单列表（支持按门店/用户筛选）超管
/api/admin/store-adminsPOST添加管理员-门店绑定 { user_id, store_id }超管
/api/admin/store-admins/:idDELETE移除单条管理员-门店绑定超管
