👥 用户角色与权限
🟢 user
普通用户
身份来源：白名单未命中，登录后默认角色
选手与观众为同一角色，通过行为（是否报名）区分。
✅ 报名参赛（选红/蓝方）
✅ 查看当前场次信息
✅ 竞猜下注
✅ 查看竞猜历史 / 排行
✅ 弹幕互动
❌ 无法访问管理员页面
🔴 admin
管理员（门店管理员）
身份来源：store_admins 白名单命中，可管理一家或多家门店
通过后台将用户 OpenID 与门店绑定产生。管理员身份互斥：不能报名参赛或参与竞猜（当前代码中 PlayerJoinService.register 未做角色校验，⚠️ 待实现）。
✅ 创建 / 管理场次（仅限白名单关联门店）
✅ 驱动赛事流程（报名→比赛→结算）
✅ 录入结算分数
✅ 查看竞猜大盘池（全量明细）
✅ 门店页右上角显示「管理」入口
❌ 不能报名参赛 ⚠️ 待实现
❌ 不能参与竞猜下注 ⚠️ 待实现
🏪 多门店管理：一个管理员可关联多家门店（store_admins 表中多条记录）。需要操作门店相关功能时（如创建赛事），前端展示门店选择器让管理员选择当前操作门店。
