🏪 选择门店页
小程序 · 选择门店页
登录成功后跳转至选择门店页（pages/index/index）。所有用户（包括管理员）均需选择门店。该页面承担三个职责：确认门店、角色路由、退出登录。
📌
导航栏 固定
标题「选择门店」，左侧返回按钮。管理员登录时（role=admin）右侧显示「管理」文字按钮
🏪
门店卡片 交互
显示门店名称「LumiLab竞技大空间」+「当前」标签；点击选中高亮边框。当前版本仅一个固定门店
🚀
进入按钮 主入口
绿色主按钮，选中门店后可点击，switchTab 跳转至赛事主页（pages/player/event）
🚪
退出登录按钮 交互
红色描边按钮，仅登录状态下显示。点击调用 userStore.logout() 清除 token + userInfo + managedStores 并跳回登录页
🛡️
「管理」入口 admin only
当 JWT 中 role === 'admin' 时，导航栏右侧出现「管理」按钮，点击 navigateTo 进入管理员操控台（pages/player/admin）。管理员仅能操作白名单关联的门店
