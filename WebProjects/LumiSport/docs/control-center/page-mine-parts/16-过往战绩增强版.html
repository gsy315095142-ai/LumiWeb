设计说明

  • 门店标识 — 顶部展示当前门店名称（如「朝阳店」），本页仅展示该门店下的参赛战绩
  • 玩法筛选栏 — 水平标签：「全部 / 🏒冰球 / 🥊拳王 / 🤺击剑」，在当前门店范围内快速切换
  • 结果标签 — 绿色「胜利」 / 红色「失败」 / 橙色「平局」
  • 积分变化行 — 展示 旧积分 → 新积分 + 差值（+N / -N），积分变化归属当前门店
  • 对阵简化 — 突出「我」的视角：「我(红方) vs 对手」，直观清晰
  • 独立子页面 — 支持滚动分页加载、玩法筛选，完整的页面体验

💡 数据来源

接口 GET /api/player/match-history?storeId={storeId}&gameType=all，支持 gameType 筛选参数，支持分页加载。
数据范围限定在当前门店，不会展示其他门店的参赛记录。
