点击「签到 +100」
→
锁定当前门店
→
后端防重复校验
→
coins += 100
写入 coin_log
→
刷新竞猜币余额

🪙 签到奖励

每次签到获得 +100 竞猜币
常量：DAILY_CHECK_IN_COINS = 100
币值归入当前门店账户

⏰ 签到规则

按 Asia/Shanghai 自然日限制
每门店每日可签到一次
幂等：重复签到返回「今日已签到」

⚠️ 并发安全

后端使用行级锁 SELECT ... FOR UPDATE + 事务保证同一用户在同一门店下不会重复签到。通过 lastCheckInYmd 字段（YYYY-MM-DD 格式）判断该门店今日是否已签到。

🏪 门店隔离说明

签到以用户 + 门店 + 日期为维度去重。用户在同一天可以在不同门店各签到一次，每个门店独立发放 +100 竞猜币至该门店竞猜币余额。
