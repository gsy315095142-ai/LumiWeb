疾速冰球 · 热身设计

示意图 交互原型

🎯热身目标：让玩家掌握接球、属性切换、投球三大核心操作，理解冰球属性（火/风/冰）的克制关系。

1. 热身步骤总览

步骤名称核心动作通过条件原型
步骤 1接球伸手接住飞来的冰球成功接球 3 次示意图
步骤 2属性切换按 A/B 键切换火/风/冰属性正确切换属性 3 次示意图
步骤 3投球瞄准靶标投出冰球命中靶标 2 次示意图
步骤 4综合练习接球 → 切换属性 → 投球完成 1 套完整流程交互

→ 步骤之间为顺序锁定：完成当前步骤后自动进入下一步。

2. 步骤详细设计

步骤 1：接球

🎯 目标
学会接住从正面飞来的冰球——这是所有后续操作的基础。
📝 VR 场景描述
玩家身处冰面竞技场中央，远处有一个自动发球机（半透明蓝色轮廓）。冰球从发球机沿直线飞向玩家，速度适中（约 2 秒到达）。冰球带有白色拖尾光效，轨迹清晰可见。顶部提示板文字：「伸出手接住飞来的冰球，接住 3 个即可通过！」
▲ 步骤 1：接球训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明；发球机发出预警音
2. 确认双手伸出就位（确认手势）提示板消失，第一颗冰球发射（3 秒延迟）
3. 接球伸手触碰飞来的冰球成功：冰球吸附到手中 + 冰蓝色光环特效 + 计数（1/3）；失败：冰球穿过 + 提示「再试一次」，自动补发
4. 完成成功接球 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 2

步骤 2：属性切换

🎯 目标
学会使用手柄按钮在火/风/冰三种属性之间切换，理解颜色对应关系。
📝 VR 场景描述
玩家手中握着一颗白色冰球，面前出现三个属性靶标：红色（火）、绿色（风）、蓝色（冰），排成一排。顶部提示板文字：「按 A 键或 B 键切换你手中冰球的属性，让它变成靶标要求的颜色！」
▲ 步骤 2：属性切换训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一个靶标亮起（随机颜色）
3. 切换按 A/B 键将冰球属性切换为靶标颜色正确：冰球变色特效 + 靶标爆裂 + 计数（1/3）；错误：提示「颜色不对，再试试」
4. 完成正确切换 3 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 3

步骤 3：投球

🎯 目标
学会瞄准并投出冰球击中目标。
📝 VR 场景描述
面前出现一个圆形靶标（类似冰球门），靶心有高亮闪烁提示。玩家手中自动获得一颗冰球。顶部提示板文字：「瞄准靶标，挥臂投出冰球，命中 2 次即可通过！」
▲ 步骤 3：投球训练场景示意图
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，冰球自动到手，靶标出现
3. 投球挥臂将冰球投向靶标命中：冰球碎裂特效 + 靶标缩小 + 计数（1/2）；未中：冰球落地消散 + 提示「再试一次」，自动补球
4. 完成命中 2 次白闪 0.6 秒 → ✅ → 暂停 2 秒 → 切入步骤 4

步骤 4：综合练习

🎯 目标
将接球、属性切换、投球串联为完整攻防流程。
📝 VR 场景描述
场景升级为正式竞技场氛围。远处发球机发射带属性颜色的冰球（红/绿/蓝随机）。面前出现对应属性的克制靶标。右侧 HUD 显示流程进度条：「接球 → 切换属性 → 投球」。顶部提示板文字：「把所有动作组合起来，完成一套完整攻防！」
🎮 交互流程
阶段玩家行为系统反馈
1. 提示阅读顶部提示板提示板展示操作说明
2. 确认双手伸出就位（确认手势）提示板消失，第一颗属性冰球发射
3. 接球接住飞来的属性冰球吸附特效 + 进度条第 1 段亮起
4. 切换将属性切换为克制靶标的颜色变色特效 + 进度条第 2 段亮起
5. 投球瞄准克制靶标投出命中爆炸特效 + 进度条全部亮起
6. 完成完成 1 套完整流程白闪 0.6 秒 → ✅ → 暂停 2 秒 → 自动循环回步骤 1

3. 视觉风格参考

VR 第一人称视角 + 游戏引擎实时渲染；视觉基调类似《Echo VR》的零重力竞技场氛围 + 赛博轻量级 UI 叠层；冰面反光与属性粒子特效为核心视觉记忆点；三属性（火/风/冰）的色彩对比要鲜明且各自独立可辨；核心设计原则：冰球飞行轨迹清晰可见、属性切换反馈即时明确、不因特效堆叠遮挡玩家视线。
