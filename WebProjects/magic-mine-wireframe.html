👤 用户中心 — 我的场次 Tab

  • 头部区域：140rpx 圆形头像 + 昵称（36rpx / 500），右上角「联系客服」+「设置」图标（position: absolute）
  • 三 Tab：「我的场次」/「我的票券」/「我的优惠券」，justify-content: space-around，选中项 #333 + 蓝色下划线（::after 6rpx #4A88FD），未选中 #A4B8C3

  • 其他场次（status ≥ 2）：右下角 CDN 状态图标（游戏中/已体验/已取消/已过期），按日期降序排列
  • 团购标记：isGroupOrder 时显示金色标签（group-order-tag，渐变橙色底 #FFD28D → #FFB347）

🎟 我的票券 Tab

点击「我的票券」Tab → 调用 /mz/getCouponList，返回散客票和团购票的混合列表。

  • 待使用：散客票 status=0 且未过期；团购票 status 0/1 且未过期且未转赠出
  • 其他票券：已使用/已过期/已核销/已失效/已赠出
  • 散客票操作：「转赠票券」（蓝色渐变按钮）+「去预约」（绿色按钮）；左滑可「小程序退票」
  • 团购票操作：「转赠团购票」+「分享给团队」+「为团队预约」（跳转 groupTeam 页）
  • 活动标签：promo_name 不为空时显示金色标签 + CDN 背景图
  • 有效期：remainingDays 红色胶囊（散客「有效期X天」/ 团购「距开场X天」）
  • 转赠遮罩：团购票已赠出时 transfer_mask 半透明黑色覆盖
  • 数据来源：/mz/getCouponList，前端按 platform + transfer_status 拆分

🏷️ 我的优惠券 Tab

点击「我的优惠券」Tab → 调用 /mz/promoCodeList，返回 promo_codes 列表。

  • 卡片样式：橙色渐变背景（待使用）/ 白色（已使用/已过期），左侧 8rpx 橙色边框
  • 右上角状态：橙金渐变「待使用」/ 灰色渐变「已使用」/「已过期」
  • 折扣类型：discount_type → 按比例折扣（8折）/ 固定金额折扣（50元折扣）/ 满减（满200减50）
  • 待使用操作：「转赠优惠券」（仅 transferable=true 时显示）+「立即使用」
  • 分享流程：转赠 → Canvas 绘制分享海报 → onShareAppMessage → 微信分享
  • 数据来源：/mz/promoCodeList，前端通过 adjustPromotionDetail 格式化折扣文案

👈 左滑退票交互

仅 platform == 'lumi' 的散客票支持左滑退票，团购票不显示。

  • 触发条件：水平滑动 > 100px（deltaX < -100），Y 轴偏移小于 X 轴时才触发
  • 卡片位移：transform: translateX(-160rpx)，右侧露出红色「小程序退票」按钮
  • 遮罩层：左滑后覆盖 rgba(0,0,0,.7) 半透明遮罩
  • 关闭条件：右滑 > 20px / 垂直滚动 / 点击其他卡片
  • 退票流程：点击红色按钮 → 确认弹窗 → /mz/refundCouponWithPayOrder → 退票成功 → 刷新列表
  • 退票限制：仅散客票（platform=lumi），团购票不支持小程序退票

联系客服

点击「联系客服」按钮 → wx.showActionSheet → wx.makePhoneCall。

  • 未选择门店时拨打通用客服电话（service_phone）
  • 已选择门店时拨打门店电话（placeLists[id].phone）
  • 无客服信息时提示「无客服信息」Toast

转赠分享机制

票券/团购票/优惠券均支持转赠（通过微信分享卡片实现）：

  • 散客票转赠：Canvas 绘制分享海报（share-coupon.jpg 模板 + 票券信息） → onShareAppMessage
  • 团购票转赠：OffscreenCanvas 绘制（share-bg3.jpg 模板 + 团队信息） → 分享标题「邀请您成为团队负责人~」
  • 团购票分享给团队：同模板，分享标题「邀请您预约场次~」，路径直接指向 groupTeam 页
  • 优惠券转赠：Canvas 绘制（discount-coupon.png 模板） → 分享标题「赠送你一张魔法学院优惠券」
  • 所有转赠只能进行 1 次，transfer_status=2 后显示「已赠出」/「已获赠」
