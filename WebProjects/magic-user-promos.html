我的优惠券

优惠券列表

调用 /mz/promoCodeList 获取用户的优惠码列表，每张优惠券显示：

  • 折扣信息：支持三种类型
    • 按比例折扣：如「8折」「6.5折」
    • 固定金额折扣：如「20元折扣」
    • 满减：如「满100减30元」
  • 有效期：显示截止日期 valid_to
  • 使用状态：未使用 / 已使用 / 已过期
  • 所属门店

优惠券赠送

优惠券支持通过微信分享赠送给好友，流程与票券赠送类似：

  • Canvas 生成分享海报（不同背景图 + 优惠券类型 + 截止时间 + 门店）
  • 分享标题：「赠送你一张魔法学院优惠券，请查收~」
团购兑换

兑换入口

用户中心顶部有团购兑换入口（图片按钮），点击后弹出兑换弹窗：

  • 输入团购码（group_code）
  • 调用兑换接口验证团购码有效性
  • 兑换成功后票券自动添加到用户的票券列表
  • 关闭弹窗后自动刷新数据
