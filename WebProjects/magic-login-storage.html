本地存储体系

存储工具函数（storage.js）

基于 wx.setStorageSync / getStorageSync 封装，提供以下功能：

函数说明
setItem(key, value)存储数据，对象/数组自动 JSON 序列化
getItem(key)读取数据，自动尝试 JSON 反序列化（仅对合法 JSON 字符串）
removeItem(key)删除指定 key
removeAllItem()清空所有本地缓存
setItemInExpire(key, value, {expire})带过期时间的存储，默认 7 天
getItemInExpire(key)读取带过期时间的存储，过期自动删除

核心存储 Key 清单

Key类型用途
access_tokenString正式 Token（API 鉴权）
tem_tokenString临时 Token（登录页使用）
expire_atStringToken 过期时间
user_infoObject用户资料（ID/头像/昵称/手机号/是否新用户）
user_loginBoolean用户是否已主动登录
user_typeString用户角色类型
currentStoreIdNumber当前选中的门店 ID
placeListsArray门店列表
service_phoneString客服电话
super_manBoolean是否超级管理员
store_staffArray门店员工角色
