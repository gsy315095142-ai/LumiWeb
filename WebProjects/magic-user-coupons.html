我的票券

数据加载

调用 /mz/getCouponList 获取用户所有票券。每张票券包含：

  • 票券名称、面值（value）、所属门店
  • 票券码（code）、平台类型（inner/group）
  • 创建时间、过期时间、剩余天数
  • 使用状态、转让状态（transfer_status）

智能分组逻辑

票券按使用状态分为「待使用」和「其他」两组，分组逻辑较复杂：

类型待使用条件其他
普通票status=0、未过期、未被转让出去status≠0、已过期、已转让
团购票status 为 0/1、未过期、非本人转出已使用、已过期、已转出
转让接收transfer_status=2 且当前用户是接收方—

排序规则：普通票按兑换时间降序，团购票按预约日期降序。

票券状态流转

待使用 (0) 待核销 (1) 已核销 (2) 已过期 (3) 已失效 (4)

票券操作

票券卡片支持多种操作：

  • 一键预约：点击「预约」跳转预约页，携带票券 ID 和面值，自动切换门店
  • 赠送票券：点击「赠送」弹出确认框，确认后通过微信分享给好友
    • 调用 /mz/markCouponTransferable 标记可转让
    • 生成 Canvas 海报（含票券类型、兑换时间、门店）作为分享封面
    • 分享链接：/pages/vouchersShareJump?share_code=1&qrcode={code}
    • 注意：赠送后自己无法再使用
  • 团购票分享：将团购票分享给团队成员预约场次
  • 团购票转赠：将整个团购票转让给他人
    • 调用 /mz/markGroupTransferable 标记可转让
    • 离屏 Canvas 生成分享海报（门店名 + 团队人数 + 预约日期）
  • 退票：左滑出现退票按钮（仅 platform=lumi）
    • 确认弹窗：退掉后不可撤销
    • 调用 /mz/refundCouponWithPayOrder
