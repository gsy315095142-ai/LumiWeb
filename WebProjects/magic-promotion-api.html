API 接口汇总

管理端接口

接口方法用途
/mz-admin/promotionPageOptsGET获取表单选项（条件/次数/类型/限制）
/mz-admin/createPromotionPOST创建优惠活动
/mz-admin/savePromotionPOST编辑保存优惠活动
/mz-admin/promotionListGET获取活动列表
/mz-admin/promotionDetailGET获取活动详情
/mz-admin/updatePromotionStatusPOST启用/禁用活动 + 配置门店/期限/有效期
/mz-admin/deletePromotionPOST删除活动
/mz-admin/promotionStatsGET活动统计数据
/mz-admin/downloadPromoExcelGET导出统计 Excel

用户端接口

接口方法用途
/mz/getCouponListGET获取用户优惠券列表（分页）
/mz/exchangeCouponPOST扫码兑换优惠券
/mz/receiveCouponPOST分享链接领取优惠券
/mz/joinGroupPOST加入团队（团票）
/mz/receiveTransferredGroupPOST接收转赠的团队票
